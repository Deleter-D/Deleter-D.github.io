scripts指定运行的脚本

```json
"scripts":{
    "test1":"命令1",
    "test2":"命令2"
    # 使用npm run test1 即可运行命令1
}
```

dependencies中指定运行时依赖

```json
"dependencies":{
    "jquery":"^3.5.1", # ^表示每次运行npm install时会更新后两位版本号
    "bootstrap":"~4.5.3", # ~表示每次运行npm install时会更新最后一位版本号
    "layui":"2.0.3" # 不加符号则固定为这个版本
}
```
